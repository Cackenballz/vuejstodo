1. open terminal
bash into
2. create laravel project
laravel new app-name
3. laravel ui
composer require laravel/ui
4. bootstrap and vue.js
php artisan ui vue --auth
5. please run "npm install && npm run dev"
npm install && npm run dev
6. make database (phpmyadmin)
utf8mb4_unicode_ci
7. compile css and js files
npm run watch
8. run a server
php artisan serv

9. create model with (m)igration (c)ontroller (r)esources (Todo is the name in the example below)
php artisan make:model Todo -mcr
